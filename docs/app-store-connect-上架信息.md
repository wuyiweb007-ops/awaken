## 二、App 描述（简体中文）

### 短描述（30 字以内，副标题）

在手机上用工字纸做每日计划、记录与省察。

### 完整描述（4000 字以内）

**觉醒笔记**把《认知觉醒》思路里的「工字型日程纸」搬到手机上：**待办、计划时间块、实际完成、省察** 四区同屏呈现，强调「计划是意图、完成是事实、省察是成长」，而不是把日程填满。

**核心能力**

- **今天**：编辑待办；将事项拆成计划时间段；在实际完成区记录真实发生（支持从计划一键带入，也支持计划外记录）；写下今日省察。
- **历史**：按日翻页回看往日工字纸，纸质翻页感浏览。
- **我的**：产品理念与用法说明；早间/晚间本地提醒开关与时间；主题（浅色/深色纸张等）；本地数据管理（如清空）。

**数据与隐私**

- 无需注册登录；日程内容默认仅保存在本机。详细说明见隐私政策：`[待补充：与 Connect 一致的 URL]`。

### What's New（首版 1.0.0）

- 首发：工字日程纸「今天 / 历史 / 我的」完整流程
- 本地提醒、主题与数据清空等基础设置

## 四、审核备注（Review Notes）

可直接与 `docs/app-store-app-review-information.md` 对齐后粘贴到 **App 审核信息 → 备注**。要点：

- **本应用不提供注册、登录、第三方登录或任何测试账号入口**；审核员安装后即可直接使用全部功能，**请勿在 Connect「登录信息」中填写测试账号**（可留空或注明 N/A）。
- **无需测试账号**；无账号，亦无账号删除流程。
- **主路径**：底部 Tab「今天」→ 编辑工字纸 →「历史」翻页 →「我的」提醒与主题、清空数据。
- **权限**：仅 **用户主动开启** 早间/晚间提醒时涉及系统**通知**；请在录屏中展示权限弹窗或按该文档说明曾授权情形。
- **外部服务**：当前无自有后端、无云同步、无内购/订阅、无广告 SDK；本地通知与本地存储为主（与 `docs/app-store-app-review-information.md` 第四节一致）。
- **录屏**：从桌面图标启动，覆盖主要功能；建议附 Resolution Center 或苹果接受的交付方式说明。

API 基址：**无**（非联网核心功能应用）。
